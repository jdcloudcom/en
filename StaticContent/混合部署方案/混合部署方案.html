Scenario Architecture

Architecture Description

JD Cloud multicloud solution provides users with the one-stop services including public cloud, hosting service, security protection, direct connection, migration service and monitoring management.
Public Cloud: Provide complete public cloud services such as Virtual Machines, cloud database, etc. so as to offer users with elastic expansion computing resource pool.
Hosting Service: Provide exclusive physical servers and hosting devices to satisfy users’ requirements for assets depreciation and elastic business expansion.
Direct Connection: Cooperate with excellent partners in the industry to provide VPN, direct connection service, as well as professional, high speed and stable connection service.
Monitoring Management: Support multiple types of resource management by the visual unified platform to provide efficient operation and maintenance management service.
Security Protection: Provide DDoS protection, Endpoint Security protection, WAF and other services covering public cloud and hosted area to guarantee users’ business security.
Migration Service: Provide Virtual Machines, Object Storage Service, database migration tool, as well as overall migration service for easy usage of cloud.
